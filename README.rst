# SQL Explorer

A modern web-based SQL Explorer built to simplify database exploration, query execution, and data analysis through an intuitive interface. The application enables users to connect to relational databases, browse database schemas, execute SQL queries, visualize results, and export datasets for further analysis.

---

## 🚀 Features

- Interactive SQL Query Editor
- Execute SQL Queries Securely
- Browse Database Schema
- Search Tables & Columns
- Query History
- Multi-Database Support
- Export Results (CSV, JSON, Excel)
- Parameterized Queries
- Read-only Database Access
- Responsive User Interface

## ⚙️ Installation

```bash
git clone https://github.com/RidhimaSharma11404/sql-explorer.git

cd sql-explorer

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
