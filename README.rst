

# 📖 Overview

SQL Explorer is a web-based database exploration platform that provides an intuitive interface for browsing database schemas, executing SQL queries, and exporting query results. It is designed to simplify database interaction for developers, students, and database administrators.

This repository contains my customized version of an open-source SQL Explorer, where I explored its architecture, configured it locally, and used it to deepen my understanding of SQL query execution, schema exploration, and database management.

---

# ✨ Features

- 🔍 Interactive SQL Query Editor
- ⚡ Execute SQL Queries
- 🗂️ Browse Database Schemas
- 📋 View Tables & Columns
- 📊 Preview Query Results
- 📁 Export Results (CSV, JSON & Excel)
- 📜 Query History
- 🔗 Multi-Database Support
- 🔒 Read-only Query Execution
- 📱 Responsive Interface

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/RidhimaSharma11404/Database-Explorer.git
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Database Migrations

```bash
python manage.py migrate
```

## Start Development Server

```bash
python manage.py runserver
```
